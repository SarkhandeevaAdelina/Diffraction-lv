Решатель дифракции на ленте (C++ версия)

Быстрый старт:
  START_HERE.bat - главное меню с выбором действий

Запуск:
  compile_and_run.bat - компиляция и запуск программы
  run_and_plot.bat - запуск с построением графиков
  run.bat - только запуск программы
  build.bat - только компиляция

Графики:
  Графики создаются автоматически при запуске run_and_plot.bat
  Сохраняются в папку cpp_version:
  - timing_analysis.png (зависимость времени от N)
  - complexity_analysis.png (сравнение с теоретической сложностью)

Результаты:
  Программа тестирует решение для N = 10, 20, 30, 40, 50
  Измеряется время построения матрицы и решения СЛАУ отдельно
  CSV данные сохраняются в timing_data.csv

Структура:
  include/ - заголовочные файлы
  src/ - исходные файлы
  build/ - скомпилированная программа
  
Параметры по умолчанию:
  a = -1, b = 1 (границы ленты)
  λ = 1 (длина волны)
  θ = 45° (угол падения)
  N = 10 (параметр усечения)
  δ = 0.1 (глубина скин-слоя)

Сравнение с C# версией:
  Запустите обе версии с одинаковыми параметрами
  Коэффициенты Чебышева должны совпадать с точностью до 4 знаков
